Description: Automated testing suite for a food ordering web application using Python, Selenium, and Pytest.

Key Features:

Page Object Model (POM): Clean and maintainable architecture.

Dynamic Data: Using uuid and datetime to generate unique test users.

CI/CD Integration: Automated test execution via GitHub Actions.

Negative Testing: Validating field boundaries (quantity limits, data types).

How to run:

Clone the repo.

Install dependencies: pip install -r requirements.txt.

Run tests: pytest.
